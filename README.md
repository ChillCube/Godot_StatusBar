# Godot_StatusBar

An addon that can be used to display custom status bars, like healthbars, xp bars, etc in your godot game.

## 📥 Installation
### Linux / MacOS
Copy and paste these commands into terminal at the root of your project:
```bash
clone-gd-addon https://github.com/ChillCube/Godot_StatusBar.git
```

Full documentation can be found in [DOCUMENTATION.md](./DOCUMENTATION.md)
